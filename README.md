## Description:
This is a MERN application that allows the user to browse for books from the [Google Books API](https://developers.google.com/books) and read basic information (title, author(s), summary,). Users can also save books to a database to be viewed again later. 

This application uses ReactJS with Bootstrap and a Mongo Database.

## Usage:
To use this application, go to [this link](https://pmw-google-book-search.herokuapp.com/).

To search for books, click the 'Search' button in the page header.
* To search for a book, type in any title or keywords in the search bar, click the 'search' button, and browse the results below.
* If you find a book that interests you, you can click the 'Add To Favorites' button, which will save that book to the database so it can be viewed again.
